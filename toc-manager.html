    Instructions

    How to publish to Super

    1. Arrange: Reorder modules (drag headers) and lessons (drag rows). Toggle Visible or edit titles. New/Stale indicators appear after merging a sitemap.
    2. Copy Head: Paste into Super → Code → Head. This embeds the lesson-manifest JSON used by your site.
    3. Copy CSS (optional, for sticky Prev/Next): Paste into Super → Code → CSS.
    4. Copy Body (optional, for sticky Prev/Next): Paste into Super → Code → Body (end).
    5. Publish in Super, then hard refresh your site.

    Keeping your order next time

    1. Save Snapshot to save your current order.
    2. Next session: Load Snapshot first, then Load Sitemap to merge new pages without losing your order/titles.
    3. Re-copy Head to update the live site.

    Tip: modules default to collapsed for faster reordering. Click ▸ to expand a module. Drag headers to move whole modules; drag rows to move individual lessons.
